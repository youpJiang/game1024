# game1024
1024 games with C++ code.

## develop environment
Ubuntu 20.04.4 LTS

# quick start
1. run  `sh build.sh`
2. run `./game1024`

## game target
move and merge the same number to reach "1024"!

## operation instruction
- move:`wsad`
- quit:`q`
